== Description ==
 
**Sell directly to verified, licensed beauty professionals.**

BeautyAList Login connects Shopify stores with verified beauty professionals, ensuring sales are made to certified experts. It offers streamlined navigation for users, customizable verification to meet specific business needs, and robust security to protect transactions. Ideal for stores aiming to enhance credibility and secure a professional clientele.

* Verified Professional Access

* Effortless User Interface

* Flexible Authentication

* Robust Security Measures

Our plugin is relying on a BeautyAList service and Licensify service.

== Installation ==

= Minimum Requirements =

* PHP 7.2 or greater is recommended
* MySQL 5.6 or greater is recommended

To do an automatic installation of the BeautyAList Login plugin, log in to your WordPress site dashboard, next to the Plugins menu. Then click "Add New" button and in the search field type "beautyalist-login" then click "Search Plugins". Click "Install Now" to install the plugin.
